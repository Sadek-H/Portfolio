## Getting Started

### Installation

```sh
git clone https://github.com/Sadek-H/portfolio.git
cd portfolio
npm install
```

### Development

```sh
npm run dev
```

Open [Live Site](https://sadekhossen.netlify.app) in your browser.

### Build

```sh
npm run build
```
